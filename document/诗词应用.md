# 诗词应用

## 接口定义

#### 数据结构

> - Network Data Holder
>   - Poem
>   - Error Info
> - Current Date

#### 接口

> - 今日诗词
>   - GET
>   - 参数：无
>   - 返回：Network Data Holder - JSON
> - 某日诗词
>   - GET
>   - 参数：day - 距2019/1/1天数
>   - 返回：Network Data Holder - JSON
>   - 注：成功后客户端更改 Current Date 值
> - 随机诗词
>   - GET
>   - 参数：无
>   - 返回：Network Data Holder - JSON
>   - 注：成功后客户端重置 Current Date 值

## 客户端功能

#### 收藏与缓存

> 存储于本地数据库中

## 数据库设计

#### 诗词

| 字段名        | 数据类型          | 属性 | 描述                         |
| :------------ | ----------------- | ---- | ---------------------------- |
| diffDay       | int               | 主键 | id为某日距2019/1/1的天数     |
| title         | varchar           |      | 标题                         |
| author        | varchar           |      | 作者                         |
| authorDynasty | varchar           |      | 年代                         |
| lines         | varchar(JSON str) |      | 诗词正文，有许多行组成的列表 |
