# 诗词应用

## 接口定义

#### 数据结构

> - Network Data Holder
>   - Poem
>   - Error Info
> - Current Date

#### 接口

> - 今日诗词
>   - GET
>   - 参数：无
>   - 返回：Network Data Holder - JSON
> - 某日诗词
>   - GET
>   - 参数：day - 距2019/1/1天数
>   - 返回：Network Data Holder - JSON
>   - 注：成功后客户端更改 Current Date 值
> - 随机诗词
>   - GET
>   - 参数：无
>   - 返回：Network Data Holder - JSON
>   - 注：成功后客户端重置 Current Date 值

## 客户端功能

#### 收藏

> 存储于本地数据库中

#### 缓存

> 数据存入数据库中
